# Atlas

Atlas is a sovereign local-first cognitive runtime and the current Hosted
implementation profile of the broader Atlas product.

## Start here

1. `ATLAS.md` — identity, direction and authority model proposed by DOC-0 RC2.
2. `AGENTS.md` — operating protocol for agent sessions.
3. `PYTHONPATH=src atlas reality --json` — current operational state.
4. `WORK_LEDGER.md` — active work and next action.
5. `docs/canon/` — machine-readable claims, sources, conflicts, authority,
   components, roadmap and coverage.

The canon is a candidate until ADR-072 is accepted and this change is merged.
Historical handoffs, packs and documents remain available as provenance but do
not become authority by filename.
