# Atlas Hosted 1.0 Roadmap — RC2 candidate

This is a convergence roadmap over the current implementation, not a claim that
Atlas must be rebuilt from zero.

- DOC-0A through DOC-0F recover canon, source absorption and traceability.
- H00 maps current authority and callers from live code/graph.
- H01-H02 define the Hosted constitution, µCore boundary and contracts.
- H03-H08 converge evidence, task, runtime, policy, models and memory.
- H09-H12 build research, domain, agent and update fabrics over those contracts.
- H13 delivers dedicated Linux and Android applications after measured stack
  prototypes.
- H14 closes Atlas Hosted 1.0 with recovery, portability, stability, security
  and operator-use evidence.
- N00 is a separate operator-opened Atlas Native research program.

The machine-readable phase graph and exit criteria live in
`docs/canon/ATLAS_ROADMAP.yaml`.
