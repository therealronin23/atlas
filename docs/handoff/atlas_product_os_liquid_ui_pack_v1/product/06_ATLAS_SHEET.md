# Atlas Sheet

Atlas Sheet is a live tabular surface connected to data, documents, APIs, AI, workflows, evidence, validation, permissions and audit.

It is not Excel cloned. It is a governed data surface.

Each cell knows:

- value;
- type;
- source;
- evidence;
- confidence;
- validation status;
- risk;
- privacy class;
- generator workflow;
- approval state;
- history;
- whether it affects outbound/submission.
