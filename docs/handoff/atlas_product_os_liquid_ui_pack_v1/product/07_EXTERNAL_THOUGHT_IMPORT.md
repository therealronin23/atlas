# External Thought Import

Atlas imports external thought to convert conversations, documents, sessions, notes and communications into reviewed memory, decisions, patterns and actions.

Sources include ChatGPT export, Claude pastes/exports, Cursor sessions, Codex/OpenHands logs, WhatsApp exports, Telegram exports, Gmail/Takeout, Obsidian vaults, Notion exports, Drive/local folders and PDFs.

Rule:

```text
Raw source preserved → extraction → memory candidate → review → memory write → audit
```

Nothing external becomes consolidated memory without review.
