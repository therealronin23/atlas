# Integration Fabric

Atlas Integration Fabric decides how to connect systems safely.

Connection Ladder:

1. Native official API
2. OAuth / scoped token / service account
3. OpenAPI / REST contract
4. AsyncAPI / event-driven API
5. Webhooks
6. MCP server/client
7. Database/file connector
8. Import/export batch
9. Browser extension / DOM bridge
10. Desktop accessibility / OS automation
11. Computer-use vision agent
12. Human-assisted/manual fallback

Rule:

```text
API-first. Computer-use last. Human confirmation always for critical actions.
```
