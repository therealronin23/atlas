# Atlas Native Business Core

Atlas should not only connect to CRM/ERP systems. It should generate a native business management core when the user has none, and link/migrate gradually when the user has old systems.

Business Core includes:

- clients;
- contacts;
- companies;
- suppliers;
- products;
- services;
- orders;
- invoices;
- receipts;
- delivery notes;
- stock;
- tasks;
- opportunities;
- budgets/quotes;
- projects/cases;
- documents;
- communications;
- payments;
- events;
- notes;
- risks;
- evidence.

Everything starts as draft before activation.
