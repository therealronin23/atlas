# Liquid Workbenches

A Liquid Workbench is an Atlas-generated work surface for a concrete objective.

Base workbenches:

- Atlas Sheet
- Workflow Canvas
- Case Board
- Project Board
- Document Review
- Evidence Graph
- Timeline / Deadline Monitor
- Communication Review
- Draft Review
- Submission Ceremony
- Writing Desk
- Research Desk
- Code / Repo Workbench
- Field Mobile Flow
- Operations Console
- Asset Library
- Calendar Planner
- Form Builder
- Report Builder
- Audit Replay
- Business Setup Workbench
- CRM Workbench
- ERP Operations Workbench
- Connection Store
