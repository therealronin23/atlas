# True Liquid Software

True liquid software in Atlas means objective-driven app generation.

Atlas does not open apps. Atlas creates the temporary or persistent environment required to solve a concrete objective.

Example:

```text
“Prepare client Martínez filing”
→ sector: gestoría/fiscal
→ workbench: Tax Filing Liquid Workbench
→ surfaces: Atlas Sheet, Document Review, Workflow Canvas, Missing Evidence, Submission Ceremony, Audit Timeline
```

Liquid software is generated from approved schemas, components and capabilities. It must not mean arbitrary unsafe code generation.
