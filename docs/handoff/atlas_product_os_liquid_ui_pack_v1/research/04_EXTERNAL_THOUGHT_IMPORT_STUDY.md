# Research 04 — External Thought Import

Atlas imports chats, documents, emails, sessions and notes as raw sources first.

Pipeline:

```text
Source Intake → Raw Preservation → Hash → Parser → Segmentation → Actor Detection → Topic/Project Detection → Extraction → Evidence Linking → Conflict Detection → Privacy Classification → Memory Candidates → Review → Memory Write → Graph Linking → Audit
```

Rule:

```text
Importar no es recordar. Importar es preparar candidatos de memoria.
```

Nothing external writes consolidated memory without review.
