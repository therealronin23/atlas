# Research 06 — Sector Taxonomy

Atlas Sector Taxonomy classifies operational worlds of work, not only economic activities.

Each sector defines objects, data, documents, tools, connectors, workflows, dangerous actions, workbenches, mobile/watch behavior, backend capabilities, risks and audit.
