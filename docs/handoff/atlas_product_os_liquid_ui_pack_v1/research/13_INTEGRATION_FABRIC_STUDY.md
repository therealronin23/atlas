# Research 13 — Integration Fabric

Atlas must discover and negotiate safe capabilities with every platform, app, server, API, provider and legacy system.

Connection Ladder:

1. Official API
2. OAuth/scoped token/service account
3. OpenAPI/REST
4. AsyncAPI/events
5. Webhooks
6. MCP
7. DB/file connector
8. Import/export batch
9. Browser extension/DOM bridge
10. Desktop accessibility/OS automation
11. Computer-use vision agent
12. Human-assisted/manual fallback

Computer-use is a high-risk fallback, never the first choice.
