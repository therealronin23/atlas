# Research 09 — Agentic Workflow Security

The LLM is not a security boundary. Prompts are not a security boundary.

Required defenses:

- Source Trust Classifier
- Privacy Classifier
- Prompt Provenance Tracker
- Capability Analyzer
- Policy Engine
- Gate Engine
- Tool/MCP/Skill Registry
- Credential Reference Store
- Secret Redactor
- Sandbox Executor
- Output Validator
- Memory Review Queue
- Audit/Merkle Logger
- Incident Recorder
- Lockdown Controller

Rule:

```text
El LLM propone. El Policy Engine decide. El Gate autoriza. El Audit Log registra.
```
