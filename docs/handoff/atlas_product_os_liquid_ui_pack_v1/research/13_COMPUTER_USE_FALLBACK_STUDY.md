# Computer Use Fallback

Computer-use exists for apps with no API, no export and no accessible database.

It must be visible, reviewable and sandboxed where possible.

Allowed: read, capture, extract, draft, assist.
Restricted: submit, sign, delete, pay, publish, file, irreversible clicks.

Every computer-use profile must declare allowed actions, forbidden actions, confirmation points, evidence capture and last verified app version.
