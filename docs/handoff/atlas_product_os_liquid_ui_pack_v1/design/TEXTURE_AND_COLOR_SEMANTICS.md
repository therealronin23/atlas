# Texture and Color Semantics

Cyan = cognition. Blue = system/read/info. Green = validated/completed. Amber = attention/review/gate. Red = error/high risk/block. Violet = external AI/cloud. White = user focus. Gray = structure/history.

Liquid = provisional. Crystal = consolidated. Filament = active relation. Fog = uncertainty. Mesh = structure. Fracture = contradiction/error. Ring = Gate. Dotted border = simulated/unconsolidated.
