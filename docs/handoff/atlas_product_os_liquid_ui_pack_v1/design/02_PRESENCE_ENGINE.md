# Atlas Product Constitution

Atlas is a local-first cognitive operating environment that converts goals into governed work.

## Product identity

Atlas is not:

- a chatbot;
- a dashboard;
- a CRM clone;
- an ERP clone;
- a workflow clone;
- a Jarvis clone;
- a thin wrapper around APIs;
- a UI skin over existing tools.

Atlas is:

```text
Objective → Sector → Data → Liquid Workbench → Validation → Governed Action → Memory
```

## Product laws

1. Local by default.
2. Cloud by necessity.
3. Audit always.
4. No silent outbound.
5. No silent certificate use.
6. No memory consolidation without review when source is external or sensitive.
7. No dangerous action without Gate.
8. No integration without capabilities.
9. No UI without state, evidence, risk and next action.
10. No dependency without policy.
