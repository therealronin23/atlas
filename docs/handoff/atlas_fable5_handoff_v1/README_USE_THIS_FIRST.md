# Atlas OS — Fable 5 Handoff Pack v1

Este paquete existe para que Fable 5 pueda construir Atlas sin perderse, dejando una base que después pueda continuar una IA menos potente.

## Qué pasarle a Fable 5

1. El repositorio actual de Atlas.
2. El build pack anterior de Atlas OS si está disponible.
3. Este paquete completo.
4. El prompt principal: `prompts/PROMPT_FABLE5_BUILD_ALL.md`.

## Objetivo

No construir una maqueta. Construir una primera versión final-compatible de Atlas OS:

- Cognitive Surface.
- Control Plane.
- Event Kernel.
- Memory OS.
- Capability Fabric.
- Governance Kernel.
- UI/UX completa con configuraciones, personalización e integraciones.
- Backend bridge real conectado progresivamente al repo existente.
- Documentación suficiente para que otra IA pueda continuar.

## Regla central

Atlas no copia productos. Atlas estudia sistemas, destila primitivas, identifica límites y reconstruye versiones superiores bajo una arquitectura soberana, auditable, extensible y viva.
