# Atlas Canon Registry

This directory is the machine-readable candidate canon for Atlas DOC-0 RC2.
It does not become repository authority merely by existing. Adoption requires
ADR-072 to be accepted and the change to be merged after isolated validation.

## Read order

1. `ATLAS_CANON.yaml`
2. `ATLAS_CLAIMS.yaml`
3. `ATLAS_PRECEDENCE.yaml`
4. `ATLAS_AUTHORITY_LEDGER.yaml`
5. `ATLAS_COMPONENT_REGISTRY.yaml`
6. `ATLAS_ROADMAP.yaml`
7. `ATLAS_COVERAGE.yaml`

`ATLAS_SOURCE_LEDGER.yaml` and `ATLAS_CONFLICT_REGISTER.yaml` preserve what
was considered, what remains incomplete and why. Handoffs and packs remain
provenance. They do not become authority by filename or age.
